Website Product Zoom

This simple-to-use and effective tool, zooms the product image to full screen on your E-commerce website.

With multiple features available in our app, multiple images can be viewed with slider option.

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Key Features

  • Image Zoom in & Zoom out
  • Zooms images to full screen
  • Image slider for multiple images
  • Restrict to zoom for poor image quality
