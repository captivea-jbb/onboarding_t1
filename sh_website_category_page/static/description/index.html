Website Category Page

Are you running the store with a large catalog of products? Wanna bring store usability to a new high level? This module is useful to show the category page for the shop. It provides easy catalog images to make the display of categories list more presentable. Grab user's attention on your store page, providing them with the immediate loadable categories listing. The default odoo category bar in the shop doesn't provide a user-friendly way to browse catalogs. Our extension allows you to use better navigation displaying all categories list on the category page. Our module is also useful to show the category name heading in the category page.

Website Product Category inside configuration.

Inside Category Check Show Category to True, then only that category visible inside "Shop By Category Page". Enter category wise description. Only those categories having images will be shown in "Shop by Category" Page.

Three different styles available in "Category Style", choose anyone to apply on "Shop by Category" Page.

Selected "Category Style": Style 1.

"Category Style": Style 1 on Mouse Hover.

Selected "Category Style": Style 2.

"Category Style": Style 2 on Mouse Hover.

Selected "Category Style": Style 3.

"Category Style": Style 3 on Mouse Hover.

Three different styles available in "Category Header Style", choose anyone to apply. When we click on any one category of "Shop by Category" Page than on redirect it shows a description of the category and chosen header style applied.

From Customize on option "Category Description," we can enable/disable category wise detail descriptions.
